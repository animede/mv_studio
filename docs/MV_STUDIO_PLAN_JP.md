# MV Studio 拡張計画メモ

## 目的
このプロジェクトは `simple_video_app` をベースに、より **MV（ミュージックビデオ）制作特化**・**ユーザー介入しやすいGUI** を備えた派生アプリとして育てるための作業用リポジトリです。

## 初期方針
- 元アプリとはリポジトリを分離して進化させる
- M2V / V2M / シーン編集体験を主軸にする
- 自動生成だけでなく、ユーザーが途中介入しやすいワークフローにする
- 中間成果物の確認・差し替え・再結合・再編集を強化する

## 想定する強化テーマ
1. **MV制作専用GUI**
   - タイムライン風UI
   - シーンカード一覧
   - 音源同期の見える化
   - トランジション・尺調整の直接編集

2. **ユーザー介入ポイントの強化**
   - シーンごとのプロンプト手修正
   - 中間画像の差し替え
   - シーン単位の再生成
   - 連結前レビュー
   - 音源に合わせたシーン尺の手動調整

3. **プロジェクト管理**
   - MVプロジェクト保存/読込
   - 楽曲ごとの作業履歴
   - エクスポート設定プリセット

4. **M2V品質改善**
   - 音源解析の詳細化
   - ビート/セクションベースのシーン分割
   - 歌詞・サビ・間奏に応じた演出分岐

## 次の実装候補
- MV向け画面設計の分離
- シーンタイムライン用データモデル追加
- M2V専用状態管理の整理
- 中間成果物ブラウザの強化
- 再結合/再同期フローの明確化
